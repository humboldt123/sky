# Sky PvP Core

[![License](https://img.shields.io/badge/license-MIT-brightgreen.svg)](https://github.com/humboldt123/sky-pvp/blob/main/LICENSE)

The super-ultra-cool-and-official Sky PvP Server Plugin!! 😊😊

![Screenshot of sky ruins with shaders](https://i.imgur.com/To3ECEh.png)

## Installation

You're going to need Java 21 and Gradle. Run `./gradlew build` and grab the jar from `build/libs/`. Or just download it from the build artifacts.

### Setup

Add the jar to your server's `plugins/` folder

## Usage

Run `/help sky` for a list of commands

## Other
This plugin was created for a private server, I just decided to open source it for the heck of it.
